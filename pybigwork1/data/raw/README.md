# 墨尔本房价

## 数据概述

melb_data 是 CSV 格式的墨尔本房产交易数据集，包含 13580 条房产交易记录与 21 个字段。数据覆盖房产位置、房产属性、交易信息等关键维度，交易价格范围为 8.5 万 - 900 万澳元（均价约 107.6 万澳元），交易日期集中在 2017 年 7-9 月。数据集主要用于房产市场分析、房价预测建模、房产投资决策辅助及城市规划参考，可满足房产相关数据分析、机器学习建模等需求。

## 字段说明

位置相关字段：`Suburb`（郊区）、`Address`（地址）、`Postcode`（邮编）、`Distance`（距市中心距离），用于定位房产空间信息。

房产属性字段：`Rooms`（房间数）、`Type`（房产类型，h 独栋屋 /u 单元房 /t 联排房）、`Bedroom2`（卧室数）、`Bathroom`（卫生间数）、`Car`（车位数量）、`Landsize`（地块面积）、`BuildingArea`（建筑面积）、`YearBuilt`（建造年份），描述房产核心特征。

交易相关字段：`Price`（交易价格）、`Method`（交易方式）、`SellerG`（中介）、`Date`（交易日期），记录房产交易关键信息。

## 数据来源

源于墨尔本房产交易市场的实际交易记录，涵盖房产交易过程中产生的位置登记信息、房产本身的物理属性数据，以及交易环节的价格、方式、中介、日期等实际备案数据。
